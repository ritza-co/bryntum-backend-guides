# Bryntum Gantt frontend starter

A basic frontend for a Bryntum Gantt. 

## Install dependencies

Run the following command to install the project dependencies:

```bash
npm install
```

⚠️ You need a Bryntum licence to install `@bryntum/gantt`. If you don't have one, install the trial package:

```bash
npm install @bryntum/gantt@npm:@bryntum/gantt-trial
```

## Run the development server

Run the Vite development server:

```bash
npm run dev
```

The server will start on `http://localhost:5173`.
